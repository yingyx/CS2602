## 简介
本仓库包含2025年秋季学期孟桂娥老师授课的*数据结构*课程的作业代码、限时作业代码和复习提纲。

推荐使用 VSCode 插件 [ACMOJ Helper](https://marketplace.visualstudio.com/items?itemName=TheUnknownThing.vscode-acmoj)。

## 文件结构

```bash
├─assignments # 作业
│  └─0-some-topic
│     └─ACMOJ-*.cpp
├─quizzes # 限时作业
├─exercises # 自主练习
└─review.pdf # 复习提纲
```
